What is this?

Remember Xfire or Raptr? This uses Discord to keep track of gametime for Steam, UPlay, Epic Games, Origin, and more.

What am I tracking?

Your status / Presence on Discord. If it changes and you're being tracked, I parse that information to determine what game you are playing as well as the game state and information available to me via presence.

Only the bot is tracking items, and the code is readily available on Github
